Complete setup guide built (March 2025) as .docx generator script. Key operational details:

**7 Phases (8-week setup):**
1. Company formation — £12 CH, SIC 68310/71129, Nov 2025 identity verification mandatory
2. RICS & professional — Firm registration + VRS (both valuers), Red Book 2025 (ESG now mandatory)
3. Insurance — PII £2-5k/yr (panels need £1m+), EL £400-800, PL £200-500
4. Compliance — AML (RICS supervised, appoint MLRO), CHP (7-day ack, 28-day response, CEDR redress), ICO £40/yr, lone worker policy
5. Technology — Rightmove Plus £150-250/mo, equipment £800-2k per set, PropVal £35/seat + £35/case
6. Panel applications — apply to ALL simultaneously once RICS registered + PII bound
7. Office — WFH initially, avoid dedicating room exclusively (CGT risk)

**Panel/AMC priority order:**
1. Method VM — 50+ lenders, 100% fee retention (no commission), apply first
2. Countrywide/Connells — highest volume, Nationwide/HSBC/Santander, £26 admin fee
3. e.surv — Virgin Money (sole), Halifax/Lloyds, mainly employed surveyors
4. Gateway Surveyors — all high-street + building societies
5. CVN Services — lender portal/API
6. SDL Surveying — network for independents, guaranteed allocation
7. VAS Group — currently closed, register interest
8. AWH/specialist/bridging — easiest entry, 70+ non-bank lenders, higher fees

**Panel requirements:** 2+ directors, 2+ MRICS/FRICS, VRS, £1m+ PII, 2yr+ PQE, sample reports, DBS

**Key lender→AMC mapping:** Nationwide→Countrywide, HSBC→Countrywide, Santander→Countrywide, Halifax/Lloyds→e.surv+L&G, Barclays→L&G+Connells+e.surv, NatWest→L&G, Virgin→e.surv(sole)

**SLAs:** Inspect within 48hrs, report same day, desktop 24-48hrs. Panels pay 30-45 days after submission.

**Financials:**
- Startup: £7-15k without vehicle, £12-30k with
- Monthly ops (3 people): £3,800-5,750
- Breakeven: 6-9 reports/month at £700
- Steady state: 50 reports/month, £35k gross, ~£30k net
- Decision to first revenue: 3-4 months
- Need 2-3 months working capital before revenue flows

**VAT critical:** Lender/panel valuations = VAT exempt (financial services). Private valuations = standard 20%. Mixed supplies = partial exemption complexity.

**How to apply:** Reference this when building PropVal panel compliance features, admin workflows, or planning firm launch timeline.
